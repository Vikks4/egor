Главная HTML CSS JS

HТМL-один из языков гипертекстовой разметки,применяется для разработки web страниц.Именно этот язык позволяет создать шаблон web страниц.Он тесно связат с такими языками программирования как CSS и JavaScript , которые стилизуют страницу и упрощают работу

На данном языке можно редактировать текст,вставлять картинки,но этого недостаточно для создания полноценного сайта.Создавать сайт ТОЛЬКО на HTML можно,но он будет не сильно функциональным,тем более это будет затрачивать много времени,именно поэтому есть дополнительные языки WEB разработки.
