JavaScript

JavaScript-язык программирования,который необходим в web разработке.С помощью него можно : создать всплывающие окна,кликеры,выполнять математические действия.В основном этот язык нужен для создания анимации и придания сайту интерактивности.

Основные команды:

console.log('Hello, World!');(Выведение текста)

let hello = 'Hi!'; console.log(hello);(создание переменной и выведение ее значения)

сonst firstName = 'Joffrey'; const greeting = 'Hello'; console.log(greeting + ', ' + firstName + '!');(создание константы и выведение ее значения)
