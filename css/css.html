Главная HTML CSS JS
CSS-язык, который отвечает за описание внешнего вида HTML-документа. Подавляющее большинство современных веб-сайтов работают на основе связки HTML+CSS,но иногда необходим JS

Для подключения CSS к HTML элементу необходимо истользовать тег link и указать название css файла

CSS используется для оформления страницы

Оформление — это цвета, шрифты, расположение отдельных блоков на странице и т.д.
